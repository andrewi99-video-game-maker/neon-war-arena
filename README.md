# Brawl Stars Clone

A top-down action game built with Pygame featuring:
- Player character with mouse-aimed shooting
- Multiple enemy AI with automatic targeting
- Projectile-based combat system
- Arena-style gameplay
- Health and ammo management

## Controls

- **W/A/S/D** - Move player
- **Mouse** - Aim direction
- **Left Click** - Shoot (while holding Space for now)
- **Space** - Fire projectiles
- **ESC** - Exit game

## Gameplay

- Defeat all enemies to win
- Collect ammo to keep shooting
- Avoid enemy projectiles
- Health regenerates slowly over time
- Each enemy has health that must be depleted

## Requirements

- Python 3.7+
- Pygame

## Installation

```bash
pip install pygame
```

## Running the Game

```bash
python main.py
```

## Game Features

- **Player Movement**: WASD controls with collision boundaries
- **Aiming System**: Mouse-based targeting
- **Enemy AI**: Enemies patrol and shoot at player
- **Projectile Physics**: Smooth projectile movement and collision detection
- **UI**: Health bars, ammo counter, enemy counter
- **Game States**: Play, Game Over, Victory
